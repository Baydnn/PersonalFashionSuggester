# How to Start the Dev Server

## Step 1: Open Terminal
Open PowerShell or Command Prompt in the `vite-project` folder.

## Step 2: Install Dependencies (if not already done)
```bash
npm install
```

## Step 3: Start the Dev Server
```bash
npm run dev
```

## Step 4: Look for Output
You should see something like:
```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## Step 5: Open Browser
Open your browser and go to: **http://localhost:5173/**

You should see the Personal Fashion Suggester app.

## If You See Errors
- Make sure you're in the `vite-project` directory
- Make sure `node_modules` exists (run `npm install` if not)
- Check the terminal for error messages
